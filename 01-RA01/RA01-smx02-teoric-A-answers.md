# RA01 -  Ex. Teòric tipus test - A

## 1. Funció principal del microprocessador

**Resposta correcta: b)** Executar instruccions i realitzar operacions aritmètiques i lògiques

Bloc 1.1.2 - Components maquinari

**Text literal del PPT:**

> “El microprocessador. És l’encarregat d’executar els programes informàtics. Executa les instruccions fent operacions aritmètiques i lògiques simples.” 

Relació directa: la resposta b és pràcticament literal.

Incorrectes:

* a) Emmagatzemar dades permanentment → el PPT diu:

  > “Els dispositius d’emmagatzematge… permeten guardar les dades… de forma permanent.” 

* c) Organitzar arxius → això surt al bloc 1.3 (sistema d’arxius) 

* d) Gestionar usuaris → no apareix associat al microprocessador en cap diapositiva.

## 2. Definició de procés

**Resposta correcta: b)** Un procés és un programa en execució

Bloc 1.3.1 - Elements del SO

**Text literal:**

> “Un procés és un programa en execució.” 

Relació directa: és literal.

 Incorrectes:
 a) “Un programa és un procés aturat” → el PPT diferencia clarament programa i procés. 

* c) “No hi ha diferència” → el PPT diu:

  > “La principal diferència entre un programa i un procés és que…” 

* d) “Només existeix al disc” → el PPT indica que el procés:

  > “ha de residir completament en memòria” 

## 3. Element que conté l’adreça de la següent instrucció

**Resposta correcta: b)** Registre comptador de programes

Bloc 1.1 - CPU / Unitat de Control

Text literal del PPT (registre PC):

> “Registre comptador de programes (PC): conté l’adreça de la següent instrucció a executar.” 

Relació directa: coincideix exactament amb la definició.

 Incorrectes:
 a) Registre d’estat → guarda flags/condicions, no adreces. 
* c) Acumulador → guarda resultats d’operacions. 
* d) ALU → executa operacions, no guarda adreces. 

## 4. Ordre del cicle de màquina

**Resposta correcta: a)** Buscar → Descodificar → Executar → Guardar resultat

Bloc 1.1 - Cicle de màquina

Text literal del PPT:

> “El cicle de màquina consta de les fases: buscar la instrucció, descodificar-la, executar-la i emmagatzemar el resultat.” 

Relació directa: l’ordre coincideix exactament.

 Incorrectes:
alsevol ordre diferent no coincideix amb la seqüència literal del PPT. 

## 5. Diferència entre RAM i ROM

**Resposta correcta: c)** La RAM és volàtil i la ROM no

Bloc 1.1 - Memòria

Text literal:

> “La memòria RAM és volàtil.” 

> “La memòria ROM és no volàtil.” 

 Relació directa: definició literal.
 Incorrectes:

* a) RAM permanent → contradicció amb “volàtil”. 
* b) ROM volàtil → contradicció amb “no volàtil”. 
* d) “No hi ha diferència” → el PPT defineix clarament les diferències. 

## 6. Bus d’adreces

**Resposta correcta: b)** Identificar el dispositiu o posició de memòria

Bloc 1.1 - Busos

Text literal:

> “El bus d’adreces identifica el dispositiu o la posició de memòria amb què es vol comunicar.” 

Relació directa: literal.

 Incorrectes:
 a) Transmetre dades → això correspon al bus de dades. 
* c) Sincronitzar → bus de control. 
* d) Executar operacions → ALU.

 Incorrectes:

## 9. Dades que romanen constants

**Resposta correcta: c)** Constants

Bloc 1.2.1 - Tipus de dades

Text literal:

> “Les constants són dades que no canvien durant l’execució d’un programa.” 

Relació directa: coincideix amb la definició.

 Incorrectes:
 Variables → “poden canviar el seu valor” 
* Intermèdies → resultats parcials
* Binàries → sistema de numeració

## 10. Dades alfanumèriques

**Resposta correcta: c)** Alfanumèriques

Bloc 1.2.2 - Codificació

Text literal:

> “Amb aquest codi es pot representar dígits del 0 a el 9, lletres majúscules… minúscules i caràcters especials.” 

Relació directa: descriu combinació de lletres, números i símbols.

 Incorrectes:
 Numèriques → només números
* Alfabètiques → només lletres
* Binàries → només 0 i 1  


## 11. En un sistema posicional, el valor d’un dígit depèn de:

**Resposta correcta: c)** El símbol i la seva posició

Bloc 1.2.2 - Sistemes de numeració

**Text literal del PPT:**

> “El valor d’un dígit en un sistema posicional depèn de la seva posició dins del nombre.” 

Relació directa: la definició indica clarament que el valor no és només el símbol, sinó també la posició.

Incorrectes:

* a) El seu color → no apareix en cap definició de sistema posicional. 
* b) El tipus de memòria → la memòria (RAM/ROM) surt al bloc 1.1, no té relació amb numeració. 
* d) El sistema operatiu → el SO es tracta al bloc 1.3, no en numeració. 

## 12. La base d’un sistema de numeració indica:

**Resposta correcta: b)** El nombre de símbols utilitzats

Bloc 1.2.2 - Sistemes de numeració

**Text literal:**

> “La base d’un sistema de numeració indica el nombre de símbols diferents que es poden utilitzar.” 

Relació directa: és pràcticament literal.

Incorrectes:

* a) Nombre total de bits → els bits són unitat mínima d’informació, no definició de base. 
* c) Mida del byte → byte = 8 bits (bloc 1.2.1), no té relació amb base. 
* d) Velocitat del sistema → concepte de rendiment, no de numeració.

## 13. El codi ASCII permet:

**Resposta correcta: b)** Representar caràcters en format binari

Bloc 1.2.2 - Codi ASCII

**Text literal:**

> “Codi ASCII. Aquest sistema utilitza una combinació de 7 o 8 bits per representar cada símbol.” 

Relació directa: “representar cada símbol” mitjançant bits implica representació binària.

Incorrectes:

* a) Convertir decimals a binari → això és conversió de bases (apartat diferent del mateix bloc). 
* c) Augmentar la velocitat de la CPU → no apareix en cap moment associat a ASCII.
* d) Gestionar processos → això surt al bloc 1.3 (processos). 

## 14. Unicode es diferencia d’ASCII perquè:

**Resposta correcta: b)** Permet representar caràcters de múltiples idiomes

Bloc 1.2.2 - Codi Unicode

**Text literal:**

> “Codi Unicode. És un codi internacional utilitzat avui en dia en la majoria dels sistemes operatius.” 

I també:

> “Permet que un programa o pàgina web… s’orienti a múltiples plataformes, idiomes o països…” 

Relació directa: internacional i múltiples idiomes.

Incorrectes:

* a) Només utilitza 7 bits → això correspon a ASCII. 
* c) Només funciona en Windows → el text diu “majoria dels sistemes operatius”. 
* d) No utilitza binari → el text parla de 8, 16 o 32 bits. 

## 15. Quina és una funció bàsica del sistema operatiu?

**Resposta correcta: b)** Gestionar recursos del sistema

Bloc 1.3 - Sistema operatiu

**Text literal:**

> “El sistema operatiu és el software bàsic de l'ordinador. Gestiona tots els recursos hardware del sistema…” 

Relació directa: la definició diu explícitament “gestiona tots els recursos”.

Incorrectes:

* a) Convertir bases numèriques → és contingut del bloc 1.2. 
* c) Executar operacions de l’ALU → això és funció del microprocessador. 
* d) Refrigerar la CPU → correspon al maquinari físic.

## 16. El kernel és:

**Resposta correcta: b)** El mòdul central del sistema operatiu

Bloc 1.3.1 - Elements del SO

**Text literal:**

> “Els sistemes operatius són formats, bàsicament, pels següents elements: processos, nucli (kernel)… ” 

Relació directa: el kernel forma part essencial del SO.

Incorrectes:

* a) Una aplicació d’usuari → el kernel no és aplicació, és part del SO. 
* c) Un perifèric → maquinari extern.
* d) Un sistema d’arxius → el sistema d’arxius és un element diferent. 

## 17. Quin dels següents és un element del sistema operatiu?

**Resposta correcta: b)** Intèrpret d’ordres

Bloc 1.3.1

**Text literal:**

> “L’intèrpret d’ordres és la interfície principal entre un usuari i el sistema operatiu.” 

Relació directa: apareix explícitament com element del SO.

Incorrectes:

* a) Bus de dades → maquinari (bloc 1.1). 
* c) ALU → part del microprocessador. 
* d) Placa base → maquinari físic.

## 18. Un directori serveix per:

**Resposta correcta: b)** Organitzar arxius dins d’una estructura jeràrquica

Bloc 1.3.1 - Sistema d’arxius

**Text literal:**

> “El sistema d’arxius permet organitzar la informació en una estructura jeràrquica de directoris.” 

Relació directa: literal.

Incorrectes:

* a) Executar programes → executar és funció del processador/SO. 
* c) Emmagatzemar instruccions en RAM → RAM és memòria (bloc 1.1). 
* d) Gestionar la CPU → funció del nucli. 

## 19. Un procés en mode usuari:

**Resposta correcta: b)** Té restriccions d’accés als recursos

Bloc 1.3 - Processos

**Text literal:**

> “Els processos d’usuari tenen accés restringit als recursos del sistema.” 

Relació directa: literal.

Incorrectes:

* a) Pot accedir directament al maquinari → això correspon al mode kernel. 
* c) És el kernel → el kernel és el nucli del SO. 
* d) No pot executar-se → sí que s’executa, però amb restriccions. 

## 20. L’intèrpret d’ordres és:

**Resposta correcta: b)** La interfície textual entre usuari i sistema operatiu

Bloc 1.3.1

**Text literal:**

> “L’intèrpret d’ordres és la interfície principal entre un usuari i el sistema operatiu.” 

Relació directa: literal.

Incorrectes:

* a) El sistema operatiu complet → el text el defineix com a element del SO. 
* c) Un tipus de memòria → no apareix en cap apartat de memòria. 
* d) Un bus del sistema → pertany al maquinari (bloc 1.1).

# RA01 -  Ex. Teòric tipus test - B

## 1. El kernel és:

**Resposta correcta: c)** El mòdul central del sistema operatiu

Bloc 1.3.1 – Elements del SO

**Text literal del PPT:**

> “Els sistemes operatius són formats, bàsicament, pels següents elements: processos, nucli (kernel)… ” 

Relació directa: el kernel forma part estructural del SO i és el nucli central.

Incorrectes:

* a) Sistema d’arxius → el PPT diferencia sistema d’arxius com element propi. 
* b) Aplicació d’usuari → el kernel no és aplicació, és part del SO. 
* d) Perifèric → és maquinari físic (bloc 1.1). 

## 2. Comandament per visualitzar contingut d’un fitxer

**Resposta correcta: d)** cat

Imatge pissarra – Operacions bàsiques

Text literal imatge:

> “cat → mostra el contingut d’un fitxer”

Relació directa: literal.

Incorrectes:

* mkdir → crea directori
* touch → crea fitxer
* ls → mostra contingut directori

## 3. La base d’un sistema de numeració indica:

**Resposta correcta: a)** El nombre de símbols utilitzats

Bloc 1.2.2

**Text literal:**

> “La base d’un sistema de numeració indica el nombre de símbols diferents que es poden utilitzar.” 

Incorrectes:

* b) Velocitat del sistema → no té relació amb numeració
* c) Nombre total de bits → bits ≠ definició de base 
* d) Mida del byte → byte = 8 bits (bloc 1.2.1) 

## 4. Diferència RAM / ROM

**Resposta correcta: c)** La RAM és volàtil i la ROM no

Bloc 1.1

**Text literal:**

> “La memòria RAM és volàtil.”
> “La memòria ROM és no volàtil.” 

Incorrectes:

* a) ROM volàtil → contradicció literal
* b) No hi ha diferència → el PPT les diferencia clarament
* d) RAM permanent → contradicció literal

## 5. Element que conté l’adreça de la següent instrucció

**Resposta correcta: b)** Registre comptador de programes

Bloc 1.1 – CPU

**Text literal:**

> “Registre comptador de programes (PC): conté l’adreça de la següent instrucció a executar.” 

Incorrectes:

* ALU → executa operacions
* Acumulador → guarda resultats
* Registre d’estat → guarda flags

## 6. Procés en mode usuari

**Resposta correcta: c)** Té restriccions d’accés als recursos

Bloc 1.3

**Text literal:**

> “Els processos d’usuari tenen accés restringit als recursos del sistema.” 

Incorrectes:

* a) És el kernel → el kernel és el nucli 
* b) Pot accedir directament al maquinari → això correspon al mode kernel
* d) No pot executar-se → sí que s’executa, però amb restriccions

## 7. Comandament que mostra contingut d’un directori

**Resposta correcta: c)** ls

Imatge pissarra

Text literal:

> “ls → mostra el contingut del directori”

Incorrectes:

* chmod → canvia permisos
* touch → crea fitxer
* cat → mostra fitxer

## 8. Funció principal del microprocessador

**Resposta correcta: d)** Executar instruccions i operacions aritmètiques i lògiques

Bloc 1.1

**Text literal:**

> “El microprocessador… Executa les instruccions fent operacions aritmètiques i lògiques simples.” 

Incorrectes:

* Organitzar arxius → sistema d’arxius 
* Gestionar usuaris → no apareix associat
* Emmagatzemar dades permanentment → dispositius d’emmagatzematge 

## 9. Dades que romanen constants

**Resposta correcta: b)** Constants

Bloc 1.2.1

**Text literal:**

> “Les constants són dades que no canvien durant l’execució d’un programa.” 

Incorrectes:

* Variables → poden canviar 
* Binàries → sistema de numeració
* Intermèdies → resultats parcials

## 10. Ordre complet del cicle de màquina

**Resposta correcta: b)** Buscar → Descodificar → Executar → Guardar resultat

Bloc 1.1

**Text literal:**

> “El cicle de màquina consta de les fases: buscar la instrucció, descodificar-la, executar-la i emmagatzemar el resultat.” 

Incorrectes:

* a) Executar abans de descodificar → contradicció literal
* c) Ordre incorrecte
* d) No coincideix amb la seqüència literal

## 11. El codi ASCII permet:

**Resposta correcta: b)** Representar caràcters en format binari

Bloc 1.2.2 – Codi ASCII

**Text literal del PPT:**

> “Codi ASCII. Aquest sistema utilitza una combinació de 7 o 8 bits per representar cada símbol.” 

Relació directa: “representar cada símbol” mitjançant bits implica representació binària.

Incorrectes:

* a) Augmentar la velocitat → en cap moment el text associa ASCII amb rendiment. 
* c) Convertir decimals → la conversió de decimals és contingut del sistema de numeració (bloc 1.2.2), no d’ASCII. 
* d) Gestionar processos → això surt al bloc 1.3 (processos), no en codificació. 

## 12. Un directori serveix per:

**Resposta correcta: c)** Organitzar arxius dins d’una estructura jeràrquica

Bloc 1.3.1 – Sistema d’arxius

**Text literal:**

> “El sistema d’arxius permet organitzar la informació en una estructura jeràrquica de directoris.” 

Relació directa: literal.

Incorrectes:

* a) Gestionar la CPU → funció del sistema operatiu/nucli. 
* b) Executar programes → executar és funció del processador/SO. 
* d) Emmagatzemar instruccions → això ho fa la memòria. 

## 13. Unicode es diferencia d’ASCII perquè:

**Resposta correcta: d)** Permet representar caràcters de múltiples idiomes

Bloc 1.2.2 – Codi Unicode

**Text literal:**

> “Codi Unicode. És un codi internacional utilitzat avui en dia en la majoria dels sistemes operatius.” 

I també:

> “Permet que un programa o pàgina web… s’orienti a múltiples plataformes, idiomes o països…” 

Relació directa: “internacional” i “idiomes” justifica la resposta.

Incorrectes:

* a) Només utilitza 7 bits → això és ASCII. 
* b) No utilitza binari → el text parla de 8, 16 o 32 bits. 
* c) Només funciona en Windows → el text diu “majoria dels sistemes operatius”. 

## 14. El bus d’adreces serveix per:

**Resposta correcta: d)** Identificar la posició de memòria

Bloc 1.1 – Busos

**Text literal:**

> “El bus d’adreces identifica el dispositiu o la posició de memòria amb què es vol comunicar.” 

Relació directa: literal.

Incorrectes:

* a) Sincronitzar instruccions → això correspon al bus de control. 
* b) Executar operacions → funció de l’ALU. 
* c) Transmetre dades → això és el bus de dades. 

## 15. Quines dades combinen lletres, números i caràcters especials?

**Resposta correcta: a)** Alfanumèriques

Bloc 1.2.2 – Codificació

**Text literal:**

> “Amb aquest codi es pot representar dígits del 0 a el 9, lletres majúscules… minúscules i caràcters especials…” 

Relació directa: descriu combinació de lletres, números i símbols.

Incorrectes:

* b) Numèriques → només números
* c) Alfabètiques → només lletres
* d) Binàries → només 0 i 1

## 16. L’intèrpret d’ordres és:

**Resposta correcta: c)** La interfície textual entre usuari i SO

Bloc 1.3.1

**Text literal:**

> “L’intèrpret d’ordres és la interfície principal entre un usuari i el sistema operatiu.” 

Relació directa: pràcticament literal.

Incorrectes:

* a) Un bus → maquinari (bloc 1.1). 
* b) Una memòria → memòria és RAM/ROM (bloc 1.1). 
* d) El sistema operatiu complet → el text el presenta com a element del SO, no el SO complet. 

## 17. Quina afirmació és correcta?

**Resposta correcta: c)** Un procés és un programa en execució

Bloc 1.3.1

**Text literal:**

> “Un procés és un programa en execució.” 

Relació directa: literal.

Incorrectes:

* a) Només existeix al disc → el PPT diu que el procés ha de residir en memòria. 
* b) Un programa és un procés en execució → definició invertida. 
* d) No hi ha diferència → el PPT diferencia clarament programa i procés. 

## 18. Funció bàsica del sistema operatiu

**Resposta correcta: c)** Gestionar recursos del sistema

Bloc 1.3

**Text literal:**

> “El sistema operatiu… Gestiona tots els recursos hardware del sistema…” 

Incorrectes:

* a) Executar operacions de l’ALU → això és funció del microprocessador. 
* b) Refrigerar la CPU → maquinari físic
* d) Convertir bases → bloc 1.2.2 

## 19. En un sistema posicional, el valor d’un dígit depèn de:

**Resposta correcta: c)** El símbol i la seva posició

Bloc 1.2.2

**Text literal:**

> “El valor d’un dígit en un sistema posicional depèn de la seva posició dins del nombre.” 

Incorrectes:

* b) Color → no apareix en cap definició
* d) Tipus de memòria → concepte del bloc 1.1

**Resposta correcta: d)** Intèrpret d’ordres

> “Els sistemes operatius són formats… processos, nucli (kernel)… sistema d’arxius… intèrpret d’ordres…”

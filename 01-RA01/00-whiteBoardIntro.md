# RA01 - INTRO  

## **Bloc 1 — Què és un sistema operatiu i per què és imprescindible**

**Objectiu a classe**
Que l’alumne entengui *què és realment* un sistema operatiu abans de parlar de res més.

**Idees clau que**

* Un ordinador **no funciona sol**
* El sistema operatiu és el **programari base**
* Fa d’intermediari entre:
  * l’usuari
  * les aplicacions
  * el maquinari

> “Sense sistema operatiu, l’ordinador és només ferro.”


## **Bloc 2 — Què fa un sistema operatiu per dins**

**Objectiu a classe**
El SO no és “una pantalla”, sinó un conjunt de funcions internes.

**Idees clau**

* Kernel (nucli)
* Gestió de processos
* Gestió de memòria
* Gestió de dispositius
* Sistema de fitxers (idea general)

> “Quan obres un programa, el sistema operatiu està treballant per dins.”


## **Bloc 3 — Tipus de sistemes operatius**

**Objectiu a classe**
**Distingir i classificar**, no només memoritzar noms.

**Idees clau**
* Sistemes monolloc
* Diferència amb altres entorns
* Sistemes propietaris vs lliures
* Exemples reals (Windows, Linux…)

> “No tots els sistemes operatius serveixen per al mateix.”


## **Bloc 4 — Com utilitza el sistema operatiu l’usuari**

**Objectiu a classe**
Relacionar la teoria amb el que es **fa cada dia**.

**Idees clau**
* Interfície gràfica
* Línia d’ordres (idea general)
* Execució d’aplicacions
* Tasques bàsiques d’ús

> “Tot el que fas amb l’ordinador passa pel sistema operatiu.”


## **Bloc 5 — El sistema operatiu dins del cicle formatiu**

**Objectiu a classe**
Donar sentit a tot el que has après.

**Idees clau **

* El SO com a base de:
  * virtualització
  * xarxes
  * dominis
  * seguretat
* Relació amb el perfil professional SMX

> “Això no s’acaba aquí: el sistema operatiu és la base de tot el curs.”
